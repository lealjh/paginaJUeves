CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para dar formato y diseñar páginas web. Permite controlar la apariencia y el diseño de los elementos HTML en un documento.

¿Cómo funciona?

CSS utiliza reglas y selectores para aplicar estilos a elementos HTML. Los estilos se definen en un archivo separado llamado hoja de estilos, que luego se enlaza al documento HTML utilizando la etiqueta ``.

Conceptos básicos de CSS

  • Selectores: Se utilizan para seleccionar los elementos a los que se les aplicarán los estilos.
  • Propiedades: Definen cómo se verán los elementos seleccionados.
  • Valores: Especifican las características detalladas de las propiedades.
  • Clases y IDs: Permiten asignar estilos específicos a elementos seleccionados.
  • Modelo de caja: Define cómo se representa visualmente un elemento, incluyendo márgenes, bordes, relleno y dimensiones.
  • Responsive Design: Técnicas para crear sitios web que se adapten y respondan a diferentes tamaños de pantalla y dispositivos.
  • Transiciones y animaciones: Permiten crear efectos de movimiento y cambio gradual en los elementos.

Ejemplo de código CSS

    
      <!DOCTYPE html>
      <html>
      <head>
        <link rel="stylesheet" href="styles.css">
        <title>Estilos CSS</title>
      </head>
      <body>
        <h1 class="titulo">Página de ejemplo</h1>
        <p class="parrafo">Este es un párrafo de ejemplo con estilos CSS aplicados.</p>
      </body>
      </html>
